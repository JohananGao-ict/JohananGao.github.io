Shell脚本

发表于 2018-12-02 | 分类于 Linux

查看shell：echo $SHELL
查看内建命令:man bash-builtins
建立多个文件touch file{001..035}
unset删除变量
wildcard：*、？、[xx]
命令代换：$(xx)、`xx`
算术代换：$(())、$[base#n]
Shell中变量的原形：${var}
转义字符：\
单双引号：’字面值’、”可扩展”

条件测试：test、[xx]
-eq、-ne:等于、不等于
-lt、-gt:小于、大于
-le、-ge:小于等于、大于等于
字符串运算符：
=字符串相等
!=字符串不相等
-z字符串长度是否为0
-n字符串长度是否不为0
逻辑运算符：
&& 逻辑and
|| 逻辑or
文件测试运算符：
-b是否块设备
-c是否字符设备文件
-d是否目录

-e是否存在文件/目录
-f是否普通文件
-g是否设置SGID位

-k是否设置粘着位
-p是否管道文件
-u是否设置SUID位

-r是否可读
-w是否可写
-s文件是否为空
-x是否可执行
0表示:真
1表示:假

分支：
if/then/elif/else/fi
;一般用作代码块标识

case/esac
;;结束

循环
for/do/done

while/do/done
break/continue

特殊变量和输入输出:
$0 相当于argv[0]
$1 $2相当于argv[1] argv[2]
$# 相当于argc-1
$@ 参数列表，可用for..in循环中
$* 同上
$? 上一条命令的exit status
$$ 当前进程号
shift 参数列表左移一个

echo
-e解析转义字符
-n 不解析转义字符

tee 输出副本文件
wc命令
-c 统计字节数
-l 统计行数
-m 统计字符数,这个标志不能与 -c 标志一起使用
-w 统计字数,一个字被定义为由空白、跳格或换行字符分隔的字符串

重定向
> 输出重定向
>> 追加
2>&1 标准错误重定向到标准输出

函数

调用函数：foo xx xx(xx表示参数)

调试shell
-n不执行，检查语法
-v一边执行一边打印到标准错误输出
-x提供跟踪执行信息
set -x启用参数
set +x禁用参数
