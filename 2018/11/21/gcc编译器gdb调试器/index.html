发表于 2018-11-21

gcc编译器

  • gcc参数：

    -E 预处理生成.i文件
    -S 编译生成汇编文件.s文件
    -c 汇编器生成目标文件.o文件
    -I 指定头文件路径
    -L 指定库路径
    -l 指定库，例如：-lm指定libm.so(math库)
    -D gcc编译时添加宏
    -O0 优化级别0不优化，最高优化级别是3(0~3)
    -Wall 输出警告信息
    -g gdb调试

gdb调试

  • gdb参数：

    l 列出源码,例如：l main.c:0 从0行开始列出main.c的内容
    b 设置断点，例如：break 2或者b 2 在第2行设置断点
    i info，例如：i b列所设置的断点
    start 开始执行，停在第一执行语句
    r 重新开始运行文件run
    n 单步调试next，也可不进入函数体内部
    c 执行到下一个断点停止continue
    s 进入到函数体内部step
    finish 从函数体内部跳出
    p i查看变量的值print
    ptype i查看变量的类型
    display i追踪变量i
    info display查看变量编号
    undisplay n取消追踪变量
    u 跳出本次循环
    d n删除断点，info break获取断点编号
    set var i=10 设置某个变量值为10
    q 退出gdb(quit)
